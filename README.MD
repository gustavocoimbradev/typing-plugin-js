# Typing Plugin JS
A plugin to create a typewriter effect

[![jsDelivr Hits](https://data.jsdelivr.com/v1/stats/packages/gh/gustavocoimbradev/typing-plugin-js/badge?period=month)](https://www.jsdelivr.com/package/gh/gustavocoimbradev/typing-plugin-js)

## Installation, Documentation, Demos & Usage Examples
https://gustavocoimbradev.github.io/docs/plugins/typing/

## Features

  * Writing time per letter typed
  * Erase time per letter erased
  * Time before erase the whole text
  * Interval time between texts typed
  * Loop (typing and erasing infinitely)
